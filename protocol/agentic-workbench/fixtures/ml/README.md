# ML Workflow Fixture Pack

This directory contains the first governed ML workflow, trust-profile, and policy-pack fixtures.

Status in this tranche:
- local training fixture
- GCP training fixture
- local trust profile
- GCP trust profile
- local policy pack
- GCP policy pack

The fixture set is designed to consume standards from `SocioProphet/socioprophet-standards-storage` and project execution into `agentplane`.
