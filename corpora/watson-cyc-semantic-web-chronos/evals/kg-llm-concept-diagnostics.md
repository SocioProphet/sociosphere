# KG / LLM Concept Diagnostics Evaluation

Status: scaffold.

Derived from:
- Conceptual Diagnostics for Knowledge Graphs and Large Language Models
- Reasoning about concepts with LLMs: Inconsistencies abound

## Goal

Measure conceptual consistency between:

- knowledge graph hierarchies
- LLM-generated answers
- ontology constraints
- retrieval and reasoning pipelines

## Candidate tasks

- subsumption consistency
- contradiction detection
- hypernym/hyponym agreement
- taxonomy completion
- KG subgraph anomaly detection
- source-quality propagation checks

## Metrics

- contradiction rate
- concept drift rate
- graph-consistency score
- evidence-backed answer ratio
- unsupported inference ratio
- source-quality propagation accuracy

## Planned components

Sherlock:
- diagnostic evidence panels
- graph extraction validation

Holmes:
- KG reasoning and proof obligations
- semantic query consistency checks

Ontogenesis:
- schema and taxonomy validation
- ontology constraint checks

Policy Fabric:
- high-risk inconsistency escalation

## Blocking conditions

No benchmark or implementation should be treated as canonical until:

- paper records are normalized
- datasets/artifacts are verified
- source quality is confirmed
