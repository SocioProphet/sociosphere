# Blocked by Legal Review

Status: queue.

This queue tracks records and implementation areas that require license, patent, or rights review before reuse or implementation.

## Cyc lineage

Blocked:
- direct ontology reuse
- assertion export
- inference-control implementation copied from proprietary material
- reuse of non-public Cyc artifacts

Allowed:
- conceptual lineage
- public-paper citation
- independent design and implementation

## Watson / DeepQA lineage

Blocked:
- direct reproduction of proprietary QA pipeline details
- implementation derived from patent claims
- unreviewed reuse of IBM artifacts

Allowed:
- evidence-ranking doctrine
- confidence propagation concepts
- public-paper architecture study

## Semantic integration / enterprise graph patents

Blocked:
- claim-derived implementation patterns
- enterprise semantic linkage workflows copied from patents

Allowed:
- benchmark comparison
- independently implemented graph workflows
- public-paper conceptual references

## Artifact reuse pending review

- CHRONOS
- ULKB Logic
- WikiCausal
- TabSketchFM
- TOPJoin
- Watson/DeepQA materials
- Cyc acquisition materials

## Exit conditions

A record leaves this queue only after:
- license verification
- access-status verification
- patent review
- design-around note if needed
- explicit reuse_mode assignment
