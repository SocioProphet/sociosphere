The lineage moves from ranking answers over text toward compiling natural language into semantic query plans over structured knowledge.

This requires:

- semantic parsing
- AMR/question graphs
- entity linking
- relation linking
- ontology grounding
- schema-aware query planning
- structured evidence traces
