# Semantic Web and Ontology Design

Status: theme synthesis.

## Core shift

The Semantic Web lineage transformed isolated expert-system knowledge into interoperable graph-based semantics.

The important transition was:

- reusable ontology patterns
- RDF/OWL-style semantics
- SPARQL-accessible knowledge
- ontology alignment
- semantic table interpretation
- graph interoperability
- formal reasoning over knowledge graphs
