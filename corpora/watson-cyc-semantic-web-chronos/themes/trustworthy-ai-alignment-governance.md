# Trustworthy AI, Alignment, and Governance

Status: theme synthesis; bibliography and benchmark normalization incomplete.

## Core shift

The lineage moves from building intelligent systems to governing them.

Alignment becomes:

- contextual
- measurable
- interactional
- provenance-bearing
- policy-bound
- auditable

The corpus shifts away from static safety prompts toward runtime governance, attribution, intervention records, drift monitoring, and contextual regulation.
