# Public-Safe Summary

Status: public-safe draft.

This summary intentionally references only records currently treated as:

- confirmed_official
- confirmed_bibliographic
- confirmed_pdf
- confirmed_artifact

The corpus captures a research lineage spanning:

- commonsense knowledge engineering
- Semantic Web and ontology design
- evidence-ranked question answering
- knowledge-graph question answering
- event-schema understanding and prediction
- causal/event knowledge graphs
- semantic table interpretation and data-lake discovery
- temporal-logic neuro-symbolics
- trustworthy AI and contextual governance

Representative confirmed works currently seeded in the corpus include:

- CHRONOS: A Schema-Based Event Understanding and Prediction System
- Conceptual Diagnostics for Knowledge Graphs and Large Language Models
- WikiCausal: Corpus and Evaluation Framework for Causal Knowledge Graph Construction
- Searching for Common Sense: Populating Cyc from the Web
- STONE: Signal Temporal Logic Neural Network for Time Series Classification

The corpus is being converted into a governed machine-readable substrate with:

- source-quality tiers
- paper and artifact registers
- graph edges and timelines
- blocked queues for unverified records
- evaluation scaffolds
- legal and provenance review paths

The target architecture separates:

- reasoning
- evidence retrieval
- schema validation
- action selection
- authorization
- audit
- bounded execution

This separation-of-duties model is designed to prevent unsafe coupling between cognition, runtime control, and actuation.

Current limitations:

- many bibliography branches remain incomplete
- artifact licenses and patents remain under review
- several KGQA and inference-control records remain blocked pending normalization
- no claim of completeness is made for any author inventory
